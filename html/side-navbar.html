Side Navigation Bar

Layout

The side navigation bar should be placed before the <div class="main-content"> so that the effects and functions works. In addition, this will cause the bar to be placed above the main content and allow its correct visualization. 
      
          <!DOCTYPE html>
          <html>
            <head>
              ...
            </head>
            <body>

              <div class="side-navbar">
                ...
              </div>

              <div class="main-content">
                ...
              </div>

            </body>
          </html>

Files

The styles and the functionalities can be found on:
  • JavaScript: side-navbar.js
  • CSS: side-navbar.css

Themes

The only theme available (for now) is the default theme. However, the following sections explain how you can create new themes.

Basic side nabvar

      
        <!--  This example uses icons from fontawesome -->

        <!-- The buttons that open / close the sidebar should have the class .sidebar-toggle-button and the data-target with the id of the sidebar to toggle -->
        <button class="default-button sidebar-toggle-button" data-target="#side-example">
          Toggle Side navbar
        </button>

        <div id="side-example" class="side-navbar default">

          <div class="side-navbar-header">
            <img src="some_link">
          </div>

          <div class="side-navbar-body">

            <ul class="items items-close-after-click">

              <li class="item">
                <i class="fa fa-umbrella" aria-hidden="true">
                This is an item with an icon at his left
              </li>

              <li class="item">
                This is an item with an icon at his right
                <i class="fa fa-umbrella" aria-hidden="true">
              </li>

              ...

            </ul>

          </div>

        </div>

Extra features

Permanent open: keeps the sidebar always open, and prevents it from being closed. Should not be used in conjunction with overlapping class. 
      
        <div class="side-navbar default is-permanent">
          ...
        </div>

Overlapping: the sidebar will be placed over the content in main-content, instead of moving it to the left. Should not be used in conjunction with is-permanent class. 
      
        <div class="side-navbar default overlapping">
          ...
        </div>
