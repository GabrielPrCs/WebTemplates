Side Navigation Bar

Bar that appears from the left side of the screen, in which you can place "items" to use as navigation within the page.

Layout

The side navigation bar should be placed after the <div class="main-content"> and be a direct child of the body tag so that the effects and functions works. 
      
          <!DOCTYPE html>
          <html>
            <head>
              ...
            </head>
            <body>

              <div class="main-content">
                ...
              </div>

              <div class="side-navbar">
                ...
              </div>

            </body>
          </html>

Files

The styles and the functionalities can be found on:
  • JavaScript: side-navbar.js
  • CSS: side-navbar.css

Themes

The only theme available (for now) is the default theme. However, the following sections explain how you can create new themes.

Extra consideration

It is only possible to have a single sidebar; if more than one is added, only the last one will be visible (the last will be the one that is below in the document).

Plane side navbar

      
        <!--  This example uses icons from fontawesome -->

        <!-- The buttons that open / close the sidebar should have the class .sidebar-toggle-button -->
        <button class="default-button side-navbar-toggle-button">
          Toggle Side navbar
        </button>

        <div class="side-navbar default">

          <div class="side-navbar-header">
            <img src="some_link">
          </div>

          <div class="side-navbar-body">

            <ul class="items">

              <li class="item active">
                <i class="fa fa-umbrella" aria-hidden="true"></i>
                This is an item with an icon at his left
              </li>

              <li class="item">
                This is an item with an icon at his right
                <i class="fa fa-umbrella" aria-hidden="true"></i>
              </li>

              ...

            </ul>

          </div>

        </div>

Extra features

Overlapping: the sidebar will be placed over the content in main-content, instead of moving it to the left. 
      
        <div class="side-navbar default overlapping">
          ...
        </div>
Close on click: closes the sidebar when a item inside is clicked. 
      
        <div class="side-navbar default">

          <div class="side-navbar-header">
            ...
          </div>

          <div class="side-navbar-body">

            <ul class="items items-close-after-click">
              ...
            </ul>

          </div>

        </div>

          </div>

        </div>

How to create a new theme

      
        <!-- Background color for the whole bar -->
        .side-navbar.default > * {
          background-color: #1C2331;
          color: white;
        }

        <!-- Background color for the part that is not covered by the upper class -->
        .side-navbar.default > .side-navbar-body > .items > li {
          background-color: #1C2331;
        }

        <!-- Color for the items when they are hover and/or active -->
        .side-navbar.default > .side-navbar-body > .items > .item:hover,
        .side-navbar.default > .side-navbar-body > .items .active {
          background-color: #3F729B;
        }
