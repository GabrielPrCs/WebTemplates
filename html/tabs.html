Tabs

Tabs are components which separate content placed in the same wrapper, but in the separate pane. Only one pane can be displayed at the time.

Layout

The tabs should be placed inside the <div class="main-content"> so that the effects and functions works. 
      
          <!DOCTYPE html>
          <html>
            <head>
              ...
            </head>
            <body>

              <div class="main-content">

                <div class="tabs">
                  ...
                </div>

              </div>

            </body>
          </html>

Files

The styles and the functionalities can be found on:
  • JavaScript: tabs.js
  • CSS: tabs.css

Themes

The only theme available (for now) is the default theme. However, the following sections explain how you can create new themes.
Each button associated with a tab must have the .tabs-link class and a data-target with the tab id to which it is associated. The .active inside a button with the .tabs-link class indicates that the tab in the data-target is the current active tab.

How to create a new theme

      
        .tabs.theme > .tabs-links {
          background-color: #1C2331;
        }
        .tabs.theme > .tabs-links > button.tabs-link {
          background-color: #1C2331;
          color: white;
        }
        .tabs.theme > .tabs-links > button.tabs-link:hover,
        .tabs.theme > .tabs-links > button.tabs-link.active {
          background-color: #3F729B;
        }
        .tabs.theme > .tabs-body {
          background-color: white;
          color: black;
        }
