Modals

The Modal is a popup window which can be used for user notifications, UI enhancements, e-commerce components and many other cases.

Layout

The modal should be placed outside of the <div class="main-content"> so that the blur effect on the background when the Modal is open works. 
      
          <!DOCTYPE html>
          <html>
            <head>
              ...
            </head>
            <body>

              <div class="main-content">
                ...
              </div>

              <div class="modal">
                ...
              </div>

            </body>
          </html>

Files

The styles and the functionalities can be found on:
  • JavaScript: modals.js
  • CSS: modals.css

Themes

The only theme available (for now) is the default theme. However, the following sections explain how you can create new themes.

Full Modal

A modal that appears from the top of the screen and is positioned approximately in the middle of the screen.

      
        
        <!-- The buttons that open / close the modal should have the class .full-modal-toggle-button and the data-target with the id of the modal to toggle -->
        
        <button type="button" name="button" class="full-modal-toggle-button" data-target="#full-modal-example">
          Open Full Modal
        </button>

        <div id="full-modal-example" class="full-modal default">
          <div class="modal-header">
            <h1>The Title!</h1>
          </div>

          <div class="modal-body">
            Lorem ipsum dolor sit amet, consectetur adipisicing elit,
            sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
            <br>
            <button type="button" name="button" class="button primary-button medium full-modal-toggle-button" data-target="#full-modal-example">
            Close Full Modal
            </button>
          </div>
        </div>

How to create a new theme

      
        .full-modal.theme > .modal-header {
          background-color: #1C2331;
          color: white;
        }

        .full-modal.theme > .modal-body {
          background-color: white;
          color: black;
        }

Bottom Modal

A Modal that appears from the bottom up on the screen. It is only possible to have one of these modals at a time.

      
        
        <!-- The buttons that open / close the modal should have the class .bottom-modal-toggle-button and the data-target with the id of the modal to toggle -->
        
        <button type="button" name="button" class="bottom-modal-toggle-button" data-target="#bottom-modal-example">
          Open Bottom Modal
        </button>

        <div id="bottom-modal-example" class="bottom-modal default">

          <div class="modal-body">
            Lorem ipsum dolor sit amet, consectetur adipisicing elit,
            sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
          </div>

          <div class="modal-footer">
            <button type="button" name="button" class="button primary-button medium bottom-modal-toggle-button" data-target="#bottom-modal-example">
              Close Bottom Modal
            </button>
          </div>

        </div>

How to create a new theme

      
        .bottom-modal.theme > .modal-body {
          background-color: white;
          color: black;
        }

        .bottom-modal.theme > .modal-footer {
          background-color: black;
          color: white;
        }
